# Syno

**Syno** is a mobile medication reminder app built with **React Native** for both **iOS** and **Android**. Inspired by my personal experience caring for a loved one during illness, I realized how important timely medication reminders are. Syno was created to make this process easier and to help others manage their health responsibly.

- **Download on iOS:** [App Store](https://apps.apple.com/es/app/syno/id6754502303?l=en-GB)  
- **Download on Android:** [Google Play](https://play.google.com/store/apps/details?id=com.barcelonacodeschool.syno)
